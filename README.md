# Relationship Lab

人际沟通演练室：帮用户准备困难对话、练习拒绝/道歉/表达需求、模拟社交场景

## What it does
- 帮用户准备边界设置、冲突修复、需求表达、高压社交等场景的沟通话术
- 提供从一句话开场到完整对话脚本的多层次输出
- 支持角色扮演模拟，帮助用户在高情绪状态下练习稳定表达

## Example scenarios

**1. 拒绝朋友的频繁请求**
> 👤 一个朋友总是找我帮忙，我不好意思拒绝，但真的不想再答应了，帮我想想怎么说
> 🤖 这更像是拒绝场景，不是解释场景。给你一个边界设置的开场："我想跟你说件事，不是不帮你，是最近我自己也需要更多精力处理自己的事"。也可以提供更柔和版和更坚定版，需要的话我陪你模拟一轮

**2. 吵架后道歉**
> 👤 和另一半吵架了，想道歉但不知道怎么说才不会显得敷衍
> 🤖 识别为冲突修复场景，帮你减少防御性。给出道歉脚本结构：平缓开场 + 说出问题 + 承担责任 + 表达意愿。可按需调整语气（更柔和/更简短）

**3. 和室友因为养猫闹矛盾**
> 👤 合租室友养了只猫，但从来不管，猫砂盆一个星期不换，满屋子臭味，我该怎么说才不伤和气？
> 🤖 "非暴力沟通"四步法：1）观察事实（"这个星期猫砂盆一直没有清理"）；2）表达感受（"我闻到味道不舒服，也担心猫咪健康"）；3）说明需要（"我们需要干净的生活环境"）；4）提出请求（"能不能商量轮流清理方案，比如一人一周？"）。如当面沟通困难，建议微信文字沟通或引入合租公约
